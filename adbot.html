🛟 PHASE 5: Customer Support (8 Support Guides)
📊
Guide 1: No Clicks
Issue: Campaigns not getting clicks
  • Check if campaigns are live
  • Explain 3-5 day learning phase
  • Set expectations: Days 4-7
  • Escalate if still no clicks after 7 days
🎯
Guide 2: No Leads
Issue: Getting clicks but no conversions
  • Check conversion tracking
  • Explain: Website converts, not ads
  • Recommend WebworX.ai
  • Keep optimizing traffic
⚡
Guide 3: Improve Performance
🔀 CHECK SERVICE FIRST
  • DIY: Use keyword thumbs up/down
  • DFY: Contact PPC specialist
➕
Guide 4: Add More Ads
🔀 CHECK SERVICE FIRST
  • DIY: Step-by-step dashboard guide
  • DFY: Send ideas to specialist
💰
Guide 5: Budget Changes
Issue: Need to adjust ad spend
  • Dashboard → Budget Settings
  • Increases: 24-48hr effect
  • Decreases: 3-5 day adjustment
  • Offer industry recommendations
📈
Guide 6: Reports Needed
Issue: Customer needs data/stats
  • Direct to Statistics tab
  • Explain key metrics
  • Executive Summary section
  • Offer to explain numbers
⛔
Guide 7: Ads Not Running
🔀 CHECK SERVICE FIRST
  • Verify payment made
  • DIY: Email support
  • DFY: Email PPC specialist
  • Common: Policy violation, tech glitch
🔐
Guide 8: Can't Login
Issue: Access problems
  • Try "Forgot Password"
  • Check spam folder
  • Clear cache/try different browser
  • Escalate: Email support with details
🚨 PHASE 6: Human Escalation
⚠️
When to Escalate
Immediate escalation triggers:
  • Bot can't resolve using guides
  • Customer explicitly requests human
  • Technical issues affecting campaigns
  • Billing/payment problems
  • Customer frustration/anger
  • Broken conversion tracking
  • Google rejected ad twice
📧
Non-Urgent Escalation
Email: hello@adbot.co.za
  • Response time: 4 hours
  • Mon-Fri, 8am-5pm SAST
  • Calculate response time for customer
  • DFY: Can also email PPC specialist
🔥
URGENT Escalation
Email with "URGENT" in subject
  • Response time: 2 hours
  • Campaigns not running
  • Billing issues
  • Major technical failures
